## 拥有功能
* 散乱 QQ 号码文本整理成列
* 企业 QQ 好友自动添加
* 普通 QQ 好友自动添加
* 营销 QQ 失效好友自动添加
* 营销 QQ 广告自动发送
* 企业 QQ 广告自动发送
* 普通 QQ 广告自动发送
* 窗口层集调整
### 调用部分
* 去除快捷方式箭头
* 恢复快捷方式箭头
* 查看系统详细信息
* 冗沉垃圾清理
* 网络dns缓存清理
* 解除拖动打开限制
* 系统检测及修复
